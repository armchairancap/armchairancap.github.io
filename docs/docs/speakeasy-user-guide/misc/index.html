Various topics and tips

This section contains tips on topics without own section.

xx Network stores Speakeasy messages for limited time​

Speakeasy's messages keeps messages for 21 days.

This is a xx Network-wide setting which instructs the gateways to delete messages after 21 days in order to prevent waste of storage space.

Currently there's no client-side settings that let one set a shorter (or longer) expiration period, but:

  • Messages can be deleted by the sender or administrator early
  • If everyone leaves a channel and no invite links or admin keys are kept, messages can be considered lost

Safety of Internet links shared in Speakeasy chat​

If you don't trust chat participants, you should not click on links shared by them in Speakeasy.

Among regular browsers, Brave Browser has a convenient feature that lets the user right-click on a link and select Open link in private window with Tor.

That doesn't give them the ability to avoid malicious sites, but at least hides their IP address the same way Tor would (or wouldn't).

Other users could copy shared links and paste them into dedicated Tor Browser instance, although that is very inconvenient. It is easier to use Brave Browser for Speakeasy.

Client-side logging and debugging​

Generic advice for Web app debugging is to try without add-ons and in Private/Incognito mode. Don't do that with Speakeasy. You should try without Private mode and with privacy features disabled for the site .

Helpful information may include your OS, browser type, browser version and logs from browser's debug tools.

Good places to get help are xx Network Discord (channel: #speakeasy) and xx Network Forum.

Detailed logs may be exported from application settings in the case developers ask for it. Because they may contain private information it's better to not share them in public without checking.

Prove wallet or asset ownership in Speakeasy​

You may send a cryptographically signed message in Speakeasy. The signature can prove ownership over a wallet or asset. See this post on how to do that in xx Network wallet.

Channel owner may use that to check if Speakeasy codenames have the minimum balance of some token to be unmuted or (out-of-band) invited to join a channel.
